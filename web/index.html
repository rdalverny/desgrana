Desgrana

From session to mix.

Just drag and drop your Behringer Wing session to get separate named mono/stereo tracks ready for your DAW to use.

Download Desgrana 1.7.0 (May 8 2026)

macOS 13 or later  ·  Apple Silicon & Intel  ·  Source code (MIT)  ·  See what's new

What it does

  • Your files come out exactly as recorded. No conversion, no precision loss, any bit depth.
  • Runs fine on a laptop in a backstage. No freeze, no RAM issues even on multi-GB sessions.
  • Only the channels you actually recorded show up. No 32 empty tracks to clean up in your DAW.
  • Your set markers land in Logic Pro, Reaper, and WaveLab; as cue chunks in the WAV, MIDI file, and CSV.
  • Channel names and stereo pairs from your Wing snapshot (KICK, SNARE, VOX…) are picked up automatically.
  • Adjust stereo pairs before extraction. No need to re-export the snapshot from the console.
  • Works entirely offline. Nothing uploaded, nothing tracked.
  • (macOS) Drop a session folder from Finder, or automate via CLI in your post-production workflow.
  • (macOS) signed and notarized.
  • Open Source.

FAQ

How do I install Desgrana?
Open the DMG archive, drag and drop Desgrana into the Applications folder, and voilà!
Is there a Windows / Linux version?

Currently, Desgrana is only available for macOS 13 and later on Apple Silicon or Intel.

Linux and Windows versions are on the roadmap.

Which consoles are supported?

Do I need the .snap or the SE_LOG.bin file?

No. Without them, Desgrana still extracts every channel into a numbered mono file. The snapshot is used to name the output files after your Wing channels and to detect stereo pairs automatically. If the .snap file is in the same session folder, it is picked up automatically.

My session is split across several WAV files: does that work?
Yes. The Wing creates a new file each time the previous one hits 4 GB (FAT32 limit). Desgrana processes all the take files in order and writes them as a single continuous output per channel.
Are session markers preserved?

Yes. Markers are embedded as standard cue chunks in every output WAV file, and also exported as a CSV, a plain-text file, and a Standard MIDI File (type 0, SMPTE timecode).

Logic Pro and most DAWs can import the MIDI file to place markers at the correct positions, independent of the project tempo.

Does it send any data online?

No, it works entirely offline. It reads your local session files and writes to your local disk. Nothing leaves your computer.

The only outbound request is an update check once a month (that you may disable) with your operating system version and current version. No usage data, no identifiers, no analytics.
